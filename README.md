#Yairt

Yet anoter Image Resize Tool

Esto es un proyecto mas para probar algunas funciones de Pillow que un proyecto serio en si,
funciona a traves de linea de comandos

## Uso

    ./yairt path [-d] [-w] [-hh] [-s] [-o]
